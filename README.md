# Go Quizz
Go quizz es una aplicación pensada por y para los estudiantes, en la cual buscamos fomentar una herramienta que si bien existía anteriormente, la hemos llevado al contexto tecnologico que vivimos actualmente, brindando al usuario tarjetas de estudio interactivas sumando los recursos multimedia que la tecnología nos permite para facilitar el proceso educativo.
